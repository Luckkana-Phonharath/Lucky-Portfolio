# Lucky Phonharath Portfolio

A modern portfolio website built with React, Vite, Framer Motion, and Tailwind CSS.

This project showcases my work, skills, and experience as a Full Stack Developer through a clean, interactive, and responsive user experience.

---

## Tech Stack

* React
* Vite
* Framer Motion
* Tailwind CSS
* Mailgun
* JavaScript

---

## Features

* Modern dark UI
* Smooth animations and transitions
* Fully responsive design
* Interactive project showcase
* Contact form integration
* Fast and optimized performance

---

## Getting Started

Clone the repository:

```bash
git clone <your-repo-url>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
